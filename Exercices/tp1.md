#  Python  Exercices du Chapitre 1  
Variables • Types • Opérations • Input • F-Strings

---

##  Introduction

Ce document contient les exercices officiels du **Chapitre 1 – Introduction à Python**.  
Ils permettent de pratiquer :

- la déclaration de variables  
- les types (`int`, `float`, `str`, `bool`)  
- les opérations mathématiques  
- l’affichage avec `print()`  
- les f-strings  
- la saisie utilisateur (`input()`)  
- les conversions (`int()`, `float()`)

Les exercices sont adaptés à des débutants et progressifs.

---

#  Exercice 1 — Profil utilisateur

Créez un programme qui déclare :

- votre prénom (texte)  
- votre âge (entier)  
- votre taille en mètres (décimal)  
- si vous aimez Python (booléen)  
- votre note préférée (décimal)

Affichez les informations sous forme de fiche + une phrase résumé.

**Exemple de sortie :**
```
=== PROFIL UTILISATEUR ===
Prénom : Kokou
Âge : 22 ans
Taille : 1.75 m
Aime Python : Oui
Note préférée : 16.5/20

Je m'appelle Kokou, j'ai 22 ans, je mesure 1.75 m, j'adore Python et ma note préférée est 16.5/20.
```

#  Exercice 2 — Opérations mathématiques

Écrivez un programme qui :

1. déclare `a = 12` et `b = 5`
2. affiche :  
   - addition  
   - soustraction  
   - multiplication  
   - division  
3. utilise des f-strings

**Exemple de sortie :**
```
12 + 5 = 17
12 - 5 = 7
12 × 5 = 60
12 ÷ 5 = 2.4
```

#  Exercice 3 — Facture avec TVA

Calculez le prix total d’achat :

- 3 livres à 15.50€
- 2 cahiers à 4.80€
- 1 stylo à 2.40€

Affichez :

1. Total HT  
2. TVA (20%)  
3. Total TTC  

**Exemple de sortie :**
```
=== FACTURE ===
Total HT : 58.50 €
TVA (20%) : 11.70 €
Total TTC : 70.20 €
```

#  Exercice 4 — Menu du jour

- 4 noms de plats (entrée, plat, dessert, boisson)

#  Exercice 6 — Conversion Celsius → Fahrenheit

Écrivez un programme qui :

1. demande une température en °C  
2. convertit en °F avec : F = C × 9/5 + 32  
3. affiche le résultat avec 1 ou 2 décimales

**Exemple :**
```
Entrez la température en °C : 25
25.0°C = 77.0°F
```

#  Exercice 7 — Calculatrice simple

Écrivez un programme qui :

1. demande deux nombres  
2. affiche :
   - la somme  
   - la différence  
   - le produit  
   - le quotient (2 décimales)

**Exemple :**
```
Entrez le premier nombre : 15
Entrez le deuxième nombre : 7
15 + 7 = 22
15 - 7 = 8
15 × 7 = 105
15 ÷ 7 = 2.14
```

#  Exercice 8 — Calculateur de pourboire

Écrivez un programme qui :

1. demande le montant de l’addition  
2. demande le pourcentage de pourboire  
3. affiche :
   - le montant du pourboire  
   - le total à payer  

**Exemple :**
```
Montant de l'addition : 45.50
Pourcentage de pourboire : 15
Pourboire : 6.83 €
Total à payer : 52.33 €
```
